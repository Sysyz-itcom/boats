# 渔舟手册

::: details 无法快速回到列表页？
点击左上角 "渔舟" 按钮返回。
:::

::: details 夜晚屏幕太亮，如何切换深色模式？
点击右上角菜单，打开 **深色模式** 设置。
:::

::: details 为什么有些章节无法进入？
仅显示 **开始探索** 的章节允许您进入，未开放的章节不会显示。
:::

::: details 在离线模式下无法打开某些页面？（404）
请您检查以下情况是否已经发生，当任意情况满足时，您都需要连网重新缓存页面数据。
- 软件已经更新并完成新版本安装。
- 您在设置中手动删除了软件的缓存。
- 软件部分页面更新。（请在右上角菜单栏中查看更新日志）
- 第三方清理程序删除了软件的缓存。
:::

::: details 收到更新但无法打开详情页？
如果您在更新弹窗中点击了 "立即更新" 您应该被重定向到渔舟软件的发布详情页。若出现如下返回信息：

```json
{
    "code": "000",
    "message": "Error",
    "details": "server is faulty. The data returned is incorrect."
}
```

请到 [社团服务页面在线状态](https://sysyz-itcom.github.io/docs/docs/club-service-running) 检查 **下载站** 是否在线。
:::
