# 渔舟手册

::: details 无法快速回到列表页？
点击左上角 "渔舟" 按钮返回。
:::

::: details 夜晚屏幕太亮，如何切换深色模式？
点击右上角菜单，打开 **深色模式** 设置。
:::

::: details 为什么有些章节无法进入？
仅显示 **开始探索** 的章节允许您进入，未开放的章节不会显示。
