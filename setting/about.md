# 关于游戏

## 关于内容
这是我们在高中无聊的课余时间和少有的闲暇时光写下的一些奇思妙想，我们将这些想法收集起来，完善了相关的故事背景，形成了 “渔舟” 。部分内容和世界观可能与一些已经发行的游戏有相似部分，对此，我们深表歉意。

::: danger ❗ 注意虚拟环境
所有的剧情、地点名词、物品名词等均为架空的虚拟定义，请勿代入现实。

部分剧情需要借助现实工具解密，但这并不意味着您可以将这些文字构建的场景带入您的生活现实世界。
:::
